Vamos a usar un modelo de regresin lineal ajustando sus parmetros mediante el mtodo del descenso del gradiente.

Nuestra funcin de error es $$E= \sum_{i=1}^{m} (y_i-(\theta_1 x_i+\theta_0))^2$$

Empecemos calculando las derivadas parciales.

\begin{equation} \begin{split} \frac{E}{\theta_0}&=\sum_{i=1}^{m} -2(y_i-(\theta_1 x_i+\theta_0)) \\ \frac{E}{\theta_1}&=\sum_{i=1}^{m} -2x_i(y_i-(\theta_1 x_i+\theta_0)) \end{split} \end{equation}

El descenso del gradiente procede en pocas, que consiste en usar el conjunto de entrenamiento para actualizar sus parmetros. Tambin interviene un parmetro que debemos de seleccionar, la tasa de aprendizaje $$, que se encarga de controlar cuanto se actualizan los valores. En cada poca actualizamos los parmetros y reduciremos el error.

\begin{equation} \begin{split} \theta_0^* &= \theta_0 -\frac{E}{\theta_0} \\ \theta_1^* &= \theta_0 -\frac{E}{\theta_1} \end{split} \end{equation}

Vamos a crear una funcin que actualice nuestros parmetros. Anotacin, el 1/n se pone para que no se dependa tanto del tamao de la muestra

In[]:
def update_parameters(x,y,b_0,b_1, a):
  dE_0 = 0
  dE_1 = 0
  n = len(x)
  for i in range(n):
    dE_0 = -2*(y[i]-b_1*x[i]+b_0)
    dE_1 = -2*x[i]*(y[i]-b_1*x[i]+b_0)
    b_0 = b_0 - a*dE_0
    b_1 = b_1 - a*dE_1
  
  return b_0, b_1

Ahora ajustaremos los parmetros de nuestro modelo actualizando b_0 y b_1 en pocas.

def train(x, y,b_0,b_1,a,epocas):
  l = np.linspace(-1, 6, 50000)
  plt.scatter(x ,y, color='blue',label = 'Muestra')
  plt.plot(l, 10*l+1, color='green', label = 'Funcin etiquetadora')
  colors = iter(('red','orange','yellow', 'pink'))
  for epoca in range(epocas+1):
    if epoca%200==0:
      f = lambda a: b_1*a +b_0  
      plt.plot(l, f(l),color = next(colors) ,label = f'Funcin predictora poca {epoca}')
      print(f'poca:{epoca}',b_0,b_1,f'MSE {mean_squared_error(y, f(x))}')
      b_0, b_1 = update_parameters(x,y,b_0,b_1, a)
    elif epoca%50==0:
      f= lambda a: b_1*a +b_0  
      print(f'poca:{epoca}',b_0,b_1,f'MSE {mean_squared_error(y, f(x))}')
      b_0, b_1 = update_parameters(x,y,b_0,b_1, a)
    else:
      b_0, b_1 = update_parameters(x,y,b_0,b_1, a)
  plt.legend()
  plt.savefig(f'linear_gradiend_descent_{epoca}.jpeg') 

  return b_0, b_1
